STAYinTouch Project

STAYinTouch is a comprehensive web-based platform designed to streamline the management of an online language school. The system provides administrators with tools to efficiently manage students, teachers, courses, and payments, while also offering an intuitive user experience for students to access lessons, track progress, and communicate with instructors.

Key features include:

  • Dual-Database Architecture: Separate databases for administrators and users ensure data integrity and security while allowing seamless information exchange.
  • Student & Teacher Management: Administrators can enroll students, assign teachers, monitor progress, and track attendance.
  • Payment Tracking System: Automates tuition fee tracking, displaying outstanding balances and due dates.
  • Course & Lesson Scheduling: Enables structured course creation, lesson assignments, and progress tracking.
  • Messaging System: Facilitates direct communication between students and administrators within the same interface.
  • User Authentication & Role Management: Secure login and role-based access control for administrators, teachers, and students.

Technologies Used:

  • Backend: Python, Flask, Flask-SQLAlchemy, Flask-Migrate, Flask-Login, Flask-Mail
  • Frontend: HTML, CSS, JavaScript
  • Database: SQLite with SQLAlchemy ORM
  • Additional Tools: Alembic for database migrations, dotenv for environment configuration

Development Highlights:

  • Designed and implemented a relational database structure optimized for educational workflows.
  • Developed an admin dashboard with real-time student progress tracking and financial insights.
  • Integrated an email notification system for automated reminders and communication.
  • Built an intuitive UI/UX experience to enhance the accessibility of educational materials.

Impact:

STAYinTouch simplifies administrative tasks, enhances learning experiences, and fosters better engagement between students and educators. Its robust architecture ensures scalability and adaptability, making it a valuable tool for online education providers.
